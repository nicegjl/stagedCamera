# stagedCamera 设计说明

**风格**：简洁、大方、有设计感  
**主题色**：与项目一致，主色 `#0a7ea4`（青蓝）作为重点操作与选中态。

---

## 设计图一览

以下三张为基于 [PRD](./PRD.md) 生成的主界面设计图，用于指导实现与评审。

### 1. 相机主界面（常规拍摄）

- **对应 PRD**：第三节 常规相机功能（拍照、录像、前后摄、闪光、变焦、画幅、相册入口等）
- **设计要点**：
  - 全屏取景区域，顶部：闪光、翻转摄像头、变焦档位（如 1x）
  - 取景区可叠加九宫格辅助线（可开关）
  - 底部：左侧最近照片缩略图（相册入口）、中央大圆形快门（主题色）、右侧画幅比例（如 4:3）
  - 拍照/录像模式切换（照片 | 视频）
- **设计图**：`../assets/design-camera-screen.png`

### 2. 人物模版库

- **对应 PRD**：第四节 人物拍照模版（模版库、分类与筛选）
- **设计要点**：
  - 顶部标题「人物模版」
  - 分类筛选：全部、单人、双人、场景、节日等（横向滑动或 Tab）
  - 模版以卡片网格展示：缩略图（姿势线框预览）+ 模版名称
  - 底部 Tab：相机 | 模版（当前在模版）
- **设计图**：`../assets/design-template-library.png`

### 3. 相机 + 模版叠加（姿势引导）

- **对应 PRD**：第四节 取景叠加、线框可调、拍摄与保存
- **设计要点**：
  - 与相机主界面布局一致，取景画面上叠加**半透明人物线框图**作为姿势参考
  - 左上或顶部可显示当前模版名称（如「模版 · 单人站姿」）
  - 用户按线框摆姿后拍摄，成片不包含线框
  - 线框透明度/大小可在设置或取景页调节（P1）
- **设计图**：`../assets/design-camera-with-overlay.png`

## 设计原则（与 PRD 一致）

| 原则     | 说明 |
|----------|------|
| 简洁     | 仅保留必要控件，取景区域为主，减少干扰 |
| 层次清晰 | 主要操作（快门、模版选择）突出，次要功能收在入口或设置 |
| 可扩展   | 模版分类、画幅、模式切换等预留入口，便于后续增加选项 |
| 一致性   | 与现有主题色、圆角、间距保持一致，便于与代码主题常量对接 |
